# Tool Website - Supabase + Next.js

Lưu ý: Đây là project đang trong quá trình phát triển. Một số TypeScript errors liên quan đến Supabase type inference đang được xử lý.

## Quick Start

1. Copy `env.template` sang `.env.local` và điền thông tin Supabase
2. Chạy migrations trong `supabase/migrations/`  
3. `npm install && npm run dev`

Xem chi tiết trong README.md
